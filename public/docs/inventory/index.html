Inventory

URL Template
http://lcboapi.com/stores/:store_id/products/:product_id/inventory

An inventory reflects the quantity of a product at a store, the reported quantity is an estimate, quantities for seasonal and discontinued products are also often inaccurate due to increased demand. This result includes store and product resources as well.

Formats

.json JSON (default format if no extension is supplied)
.js JavaScript (for use with JSON-P, requires callback parameter)
.csv Comma Separated Values
.tsv Tab Separated Values

Fields

product_id Product ID
store_id Store ID
is_dead True if this inventory refers to a dead store and/or product
quantity Reported quantity on hand of the product ID at store ID
updated_on Reported date that quantity was updated
updated_at Time that this inventory item was updated
product_no Product ID [Deprecated]
store_no Store ID [Deprecated]
